### 3. Check Link Quality (10 minutes)

**DoFollow vs NoFollow Check:**

Right-click on your link → Inspect Element → Look for:
- `<a href="https://mikahsmobiledetailingsc.com">` = **DoFollow** ✅ (passes SEO juice)
- `<a href="..." rel="nofollow">` = **NoFollow** ⚠️ (less SEO value)
- `<a href="..." rel="ugc">` = **User Generated Content** ⚠️ (less SEO value)

**High-value DoFollow Links (prioritize these):**
- Dev.to articles
- About.me profile
- Linktr.ee/Linkin.bio
- Issuu documents (sometimes)

## ⚠️ IMPORTANT RULES

### DO:
✅ Create quality content on each platform
✅ Use different anchor text variations
✅ Diversify platforms (not all from one site)
✅ Build links gradually (10-30/month is natural)
✅ Link to different pages (homepage, services, blog posts)

### DON'T:
❌ Buy links from Fiverr/black hat services
❌ Use exact same text everywhere
❌ Build 100 links overnight (Google penalty risk)
❌ Only link to homepage (looks unnatural)
❌ Ignore link quality (1 high-quality > 10 low-quality)
