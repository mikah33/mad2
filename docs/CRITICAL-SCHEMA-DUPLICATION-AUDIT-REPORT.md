## 🔍 TECHNICAL ROOT CAUSES

### Issue #1: Double Build Pipeline
```typescript
// BUILD TIME: generate-all-pages-html.ts (lines 70-112)
const schemaScript = `<script type="application/ld+json">
  ${JSON.stringify(enhancedSchema, null, 2)}
</script>`;
html = html.replace('</head>', `${schemaScript}\n  </head>`);

// RUNTIME: CeramicCoatingPage.tsx (lines 77-79)
<Helmet>
  <script type="application/ld+json">
    {JSON.stringify(schemas)}
  </script>
</Helmet>
```
**Result:** Both scripts run → 2x LocalBusiness schemas on every service page

### Issue #2: HomePage Has Triple Organization Risk
```typescript
// STATIC: dist/index.html (lines 46-84)
Organization schema

// REACT: App.tsx (lines 40-50)
const organizationSchema = generateOrganizationSchema(...)

// REACT: App.tsx (lines 33-37)
const localBusinessSchema = generateLocalBusinessSchema(...)
```
**Result:** 1 Organization (static) + 1 Organization (React) + 1 LocalBusiness (React)

### Issue #3: React Components Always Add LocalBusiness
**Every service page does this:**
```typescript

```
This adds LocalBusiness even though `generate-all-pages-html.ts` already added it!
